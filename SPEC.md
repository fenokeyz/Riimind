# Riimind

## Goal

Turn any natural language message into a calendar event in under 10 seconds.

## MVP Flow

1. User pastes text or shares text from another app.
2. App sends text to Gemini.
3. Gemini extracts:
   - title
   - description
   - date
   - time
   - location
4. User reviews and edits.
5. Event is added to the device calendar.

## MVP Features

- Paste text
- Clipboard detection
- Share from other apps
- Gemini parsing
- Event preview
- Device calendar integration

## Non Goals

- Login
- Backend
- Cloud sync
- Firebase
- Ads

## Design

- Material 3
- Clean
- Minimal
- Android first
